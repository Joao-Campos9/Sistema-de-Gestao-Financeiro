# Sistema-de-Gestao-Financeiro
Esse repositório foi todo projetado por mim, para uma atividade avaliativa da PUCPR. O projeto tem como objetivo aplicar os conceitos de lógica de programação com python, e desenvolvimento web com o framework Flask, criando interfaces interativas e funcionais

## Tecnologias usadas
- Python
- Flask
- JSON (para permanência de dados)
- HTML / CSS
- Jinja2
- Bootstrap
- Chart.js

## Como rodar o projeto
1. Extraia a pasta .zip
2. Abra a pasta do projeto dentro do VS Code
3. Digite o comando para baixar as dependências do projeto
   pip install -r requirements.txt
4. Execute o arquivo main.py
5. No navegador, acesse o endereço local localhost:5000
